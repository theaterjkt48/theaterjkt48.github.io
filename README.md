# 🎭 Theater48

**Website informasi show & member JKT48 Theater**

🌐 **Live Site:** [theater48.github.io](https://theater48.github.io)

---

## ✨ Fitur

- 🔴 **Live Show Banner** — notifikasi otomatis saat show sedang berlangsung
- 👥 **Daftar Member** — lengkap dengan generasi, tim, dan jumlah penampilan
- 🎵 **Filter Setlist** — Ramune, TTE, Pertaruhan, Pajama Drive
- 🔍 **Pencarian Member** — real-time search by nama
- 📅 **Jadwal Show** — schedule pertunjukan terbaru
- 📊 **Statistik Penampilan** — ranking member berdasarkan jumlah show
- 🌙 **Dark / Light Mode** — tampilan bisa disesuaikan
- ⚡ **Realtime Update** — data diperbarui otomatis via Supabase

---

## 🗂️ Halaman

| File | Deskripsi |
|------|-----------|
| `index.html` | Halaman utama — daftar member & filter setlist |
| `schedule.html` | Jadwal show mendatang |
| `liveshow.html` | Detail show yang sedang / akan berlangsung |
| `member.html` | Profil detail per member |
| `riwayat.html` | Riwayat show yang telah lewat |

---

## 🛠️ Tech Stack

| Teknologi | Kegunaan |
|-----------|----------|
| HTML / CSS / JS | Frontend utama (vanilla, tanpa framework) |
| [Supabase](https://supabase.com) | Database & realtime subscription |
| GitHub Pages | Hosting gratis |

---

## 🚀 Cara Menjalankan Secara Lokal

```bash
# Clone repository
git clone https://github.com/theater48/theater48.github.io.git

# Masuk ke folder
cd theater48.github.io

# Buka di browser
open index.html
```

> Catatan: Beberapa fitur seperti data member & realtime membutuhkan koneksi ke Supabase.
